# Starbust Token Calculator

Starbust Token Calculator is a lightweight utility designed to help developers estimate and compare input token consumption when querying databases. 

The tool provides a side-by-side analysis of **Starbust-optimized** queries versus **Standard LLM-to-SQL** approaches, helping you optimize for both cost and performance.

## 🚀 Features
* **Token Estimation:** Accurate counts for prompt overhead and schema metadata.
* **Cost Comparison:** See the projected savings when using Starbust's architecture.
* **Customizable Schemas:** Input your own table structures to see how complexity affects token usage.

## 📊 Why Starbust?
Standard LLM querying often requires passing massive amounts of schema metadata or "few-shot" examples in every prompt, leading to high token costs. Starbust optimizes this flow to minimize the input context without losing query accuracy.
